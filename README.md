# Boosted Builders

A tiny static matched betting helper for UK reload offers, boosted Bet Builders and awkward equivalent-market lays.

## What it calculates

- Equivalent market lays, for example backing `Draw + BTTS + Under 3.5 Goals` and laying `Correct Score 1-1`.
- Direct boosted odds lays.
- Winnings boosts, such as `50% extra winnings`.
- Free bet SNR lays.
- Refund-if-loses offers using a cash-equivalent retention percentage.

## Deploy on GitHub Pages

1. Open this repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.

The site should then publish at:

```text
https://rcampbell30.github.io/BoostedBuilders/
```
